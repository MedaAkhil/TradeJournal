# Trade Journal 📊

A complete trading journal application to track daily trades, analyze performance, and store screenshots.

## Features ✨

- **Daily Journal** - Log multiple trades per day with screenshots
- **Calendar View** - Color-coded calendar showing profitable/losing days
- **Analytics Dashboard** - Charts and metrics for performance analysis
- **Backup & Restore** - Export/import your entire journal
- **Screenshot Support** - Upload or paste images directly
- **Responsive Design** - Works on mobile and desktop

## Live Demo 🚀

[https://trade-journal-pro-e0ee0.web.app](https://trade-journal-pro-e0ee0.web.app)

## Tech Stack 💻

- HTML5, CSS3, JavaScript
- Firebase Firestore (Database)
- Firebase Storage (Images)
- Firebase Hosting
- Chart.js for analytics
- Font Awesome for icons

## Local Development 🛠️

1. Clone the repository
2. Install Firebase CLI: `npm install -g firebase-tools`
3. Run locally: `firebase serve`
4. Deploy: `firebase deploy`

## Usage 📖

1. Navigate to any date using the date picker
2. Click the floating **+** button to add trades
3. Upload screenshots or paste from clipboard
4. View calendar for monthly overview
5. Check dashboard for performance analytics
6. Regularly export backups for safety
